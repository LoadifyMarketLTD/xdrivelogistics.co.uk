# Image Assets for XDrive Landing Page

This directory contains placeholder SVG images for the XDrive landing/sign-in page.

## Current Files (Placeholders)

- `logo.svg` - XDrive logo placeholder
- `hero.svg` - Hero/night motorway background placeholder  
- `thumb1.svg`, `thumb2.svg`, `thumb3.svg` - Feature thumbnail placeholders

## Production Assets

**To use production assets**, replace these SVG files with actual image files:

- `logo.png` or `logo.svg` - The official XDrive Logistics logo
- `hero.jpg` - Night motorway/map hero image (recommended: 1920x1080px)
- `thumb1.jpg`, `thumb2.jpg`, `thumb3.jpg` - Feature preview thumbnails (recommended: 400x300px)

Update the `src` attributes in `index.html` to match the file extensions of your production assets (e.g., change `.svg` to `.jpg` or `.png`).

## Notes

- Current SVG placeholders are provided for development and demonstration purposes
- The HTML and CSS are designed to work with both SVG and raster image formats (PNG, JPG)
- All images should maintain good contrast against the dark theme background
